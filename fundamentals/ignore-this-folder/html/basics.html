Q-6. Write html code to print the following try the following formatting tags?

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Small text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

Difference between <b> and <i>: <b> and <i> defines bold and italic text, but <strong> and <em> means that the text is "important".

  • <p> is used for all elements as a wrapper.
  • eg: <p><b>This text is bold</b></p>


Quotation & Citation

Brief Intro: Quotation is defined as statement or saying "said or written" by someone or some organization. Citation is the process of giving someone "credit".

Q-7. Write html code to print in following way using tags prescribed below?

  • <p> is used for all each paragraph.
  • <q> is used for 2nd paragraph and <strong> is used to highlight management system.
  • <br> is used for space between 2nd paragraph & 3rd paragraph.
  • In 4th + 5th paragraph (here is - one paragraph & main quote - another paragraph), Use <blockquote> for wrapping the quotation, keep cite ATTRIBUTE of the blockquote as "http://www.worldwildlife.org/who/index.html". Use <abbr> tag to wrap WWF & keep title attribute of WWF to "World Wildlife Fund" inside both paragraphs. Use <cite> for wrapping WWF's website.
  • In 6th + 7th paragraph (Contact us - one paragraph & address - another paragraph), Use <address> tag to wrap all the elements, <strong> for highlighting company name & <br> for all the line breaks & spaces. <abbr> for WWF is optional.
